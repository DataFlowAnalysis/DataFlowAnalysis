## 📝 Short Description
This DFD represents the documented structure and data flows in the Kulturpass DE documentation. The main parts of the DFD are the Frontend Layer, mainly composed of the Mirakl UI and the Backend Layer, mainly composed of the Customer Data Cloud (CDC) and Mirakl. The DFD shows the data flows on the seller registration process with ELSTER, user identification with eID and geospatial search concepts.

## 🏷️ Label description
- ### 🗂️ Data Labels:
    - __ConsumerInfo__:
        - EId: electronic id of a user
        - IdToken: Generated IdToken related to EId of the user
        - UserId: User identifier inside the system
        - Account: Account of a user
        - tcToken: Token generated by the Storefront App/Web
        - PIN: User PIN
        - Pseudonym: User pseduonym in the system
        - EIddata: Data of the electronic id card
        - CustomerIData: Data of the customer
        - ZipCode: Zipcode of the customer
    
    - __Requests__:
        - SAML: Request is a SAMl request
        - PIN: Request is for a PIN
        - ForceUpdate: Request is a force update request
    
    - __Response__: 
        - SAML: Response is a SAML response
        - Confirmation: Response is a confirmation

    - __CulturalProviderInfo__:
        - Email: Mail from the cultural provider
        - ShopName: Name of the shop/business of the cultural provider
        - ELSTERCertificate: Certificate of the business for ELSTER
        - Password: Password of the cultural provider
        - CompanyInfo: Information about the company/business
        - ZipcodesCsv: Zipcodes about location of its business
        - GeocodedZipcodes: Zipcodes generated by the geospatial search concept

    - __ELSTERConnector__:
        - Session: Session in the ELSTERConnector
    
    - __CDCData__:
        - Sellers: Data of sellers in the system

    - __Mirakl__:
        - SucessMessage: Mirakl success message
        - ErrorMessage: Error message of Mirakl
        - Offers: Business offers in Mirakl
        - Sellers: List of sellers of the businesses

- ### 🏷️ Node Labels:
    - __Component__:
        - StorefrontApp/Web: The component is/is part of the Storefront App/Web
        - EIdConnector: The component is/is part of the eIdConnector
        - AusweisApp2: The component is/is part of the AusweisApp2
        - CDC:  The component is/is part of the CDC
        - CommerceAPI: The component is/is part of the CommerceAPI
        - ELSTERConnector: The component is/is part of the ELSTERConnector
        - ELSTERIdP: The component is/is part of the ELSTERIdP
        - Mirakl: The component is/is part of the Mirakl
        - SellerReplicator: The component is/is part of the SellerReplicator
        - CommcerceCloud: The component is/is part of the Commerce Cloud
        - HANA: The component is/is part of the HANA
