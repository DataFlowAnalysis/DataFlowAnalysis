The xDECAF data flow diagram analysis constraint framework provides several frameworks to interact with the analysis:

The Online Editor that allows for modelling Data Flow Diagrams online and provides functionality to analyze them as well.

The Command Line Interface that allows you to run the analysis within an binary locally.

Or the analysis frameworks for both Data Flow Diagrams and the Palladio Component Model

Lastly, the provided Converter allows for conversions between several models and allows using more models for the analysis.
