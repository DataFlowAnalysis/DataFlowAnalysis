Overview

Download â€‹

We provide multiple ways to download and use the data flow analysis framework. The easiest way to get started is downloading our ready-to-use Eclipse Product. Alternatively, all major artifacts are available on our Eclipse Updatesite to be directly installed into the Eclipse Modeling Framework. Last, if you only want to model and analyze simple data flow diagrams, you can use our online editor without any installation required. Afterwards, visit Getting Started to learn more about using the analysis.

Eclipse Updatesite â€‹

We provide an Eclipse Updatesite with all major artifacts ready to be installed in the Eclipse Modeling Framework. The updatesite comprises both the latest release artifacts and a nightly build.

Release (add this URL to your Updatesites in Eclipse):

https://dataflowanalysis.github.io//updatesite/release

Nightly (add this URL to your Updatesites in Eclipse):

https://dataflowanalysis.github.io//updatesite/nightly
