## Overview
The Data Flow Analysis can work with multiple types of models and different editors.
### Models
For working with Palladio (PCM) models, please see [here](pcm/intro.md)

For working with Data Flow Diagrams (DFDs), please see [here](dfd/intro.md)

### Modelling and Development Environments
Our fully fledged Web Editor can be used to model [Data Flow Diagrams](dfd/data-flow-diagrams.md) and analyze them, directly in the browser without needing to run any code!
The results of your analysis can be viewed in the editor as well, so no need to download any tools!
If you seek information for working with the [Web Editor](https://editor.dataflowanalysis.org/), please see [here](webeditor/intro.md)

Our data flow analysis is currently built on Eclipse, with Eclipse plugins and a [product](https://github.com/DataFlowAnalysis/product/releases/latest) available for download.
You are able to edit both [Data Flow Diagrams](dfd/data-flow-diagrams.md) and [Palladio](pcm/models.md) in Eclipse.
For running the analysis locally using our Java Framework, please take a look at the [Running Locally](running-locally.md) section of this wiki.

If you are working, or want to work, with out Eclipse product, you can find information about out development setup [here](eclipse/intro.md)

If you are working, or want to work, with IntelliJ, you can find relevant information about setup and usage [here](intellij/intro.md)

We also offer a Command Line Interface (CLI) for working with the Data Flow Analysis.
For a guides and documentation to use the command line interface, see [here](cli/intro.md)
